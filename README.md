# Marketing Mix Regression Analysis (R)

This project applies simple and multiple linear regression to analyze the effectiveness of marketing channels (Radio, TV, In-Store Promotion, and Discount) on product sales using real-world style marketing mix data.

The goal is to identify which channels truly drive sales and how different budget strategies affect expected performance.
